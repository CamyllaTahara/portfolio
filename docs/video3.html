Calculadora em Java

Este projeto consiste em uma calculadora gráfica desenvolvida em Java Swing, capaz de realizar operações matemáticas avançadas de forma intuitiva e eficiente. Funcionalidades: Permite calcular a potência de um número, sua raiz quadrada e cúbica, por fim, exibe o módulo de um número, transformando negativos em positivos.
